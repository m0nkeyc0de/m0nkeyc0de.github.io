List of handy tools and other resources coming accross my way.

## On the web

* [Matt Brown's Github](https://github.com/nmatt0/) : IoT hardware hacker (also has a [Youtube Channel](https://www.youtube.com/@mattbrwn)).
* [David Bombal's Youtube channel](https://www.youtube.com/@davidbombal) : lots of interesting stuff about security topics.
* [Certified Wireless Network Professionals](https://www.cwnp.com/) : qualitative and vendor-neutral Wi-Fi (and IoT) education and certfications.

## Software

* [Ventoy](https://www.ventoy.net/) : Live USB bootloader with allows to chose from which ISO file you want to boot from.
* [ImHex](https://imhex.werwolv.net/) : Very nice hex editor.

## Hardware

* [**Alfa Network AWUS036AXML**](https://www.alfa.com.tw/products/awus036axml) : a Wi-Fi 6E adapter, based on a MediaTek MT7921AUN chipset, and which works fine.
