On the web#

  • Certified Wireless Network Professionals : qualitative and vendor-neutral Wi-Fi (and IoT) education and certfications.

Software#

  • Ventoy : Live USB bootloader with allows to chose from which ISO file you want to boot from.
  • ImHex : Very nice hex editor.

Hardware#

  • Alfa Network AWUS036AXML : a Wi-Fi 6E adapter, based on a MediaTek MT7921AUN chipset, and which works fine.
