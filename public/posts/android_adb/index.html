Android Debug Bridge tricks

Posted on Sep 22, 2024

ADB tricks

Commands

List installed applications

adb shell pm list packages

Get application PID

adb shell pidof -s com.application.name

Logs for given PID

adb logcat --pid=8064

Screenshot

adb exec-out screencap -p > "$(date).png"
